---
name: No usar Chrome DevTools para verificar UI
description: El usuario prefiere verificar los cambios de UI él mismo, sin que Claude use Chrome DevTools
type: feedback
---

No usar Chrome DevTools MCP para tomar screenshots o interactuar con el navegador después de hacer cambios de UI. El usuario prefiere revisar los cambios visualmente por su cuenta.

**Why:** El usuario lo indicó explícitamente cuando se usaban las herramientas de Chrome para verificar cambios CSS/JS.

**How to apply:** Después de hacer cambios de CSS o JS en la UI, describir los cambios realizados y dejar que el usuario los verifique en el navegador.
